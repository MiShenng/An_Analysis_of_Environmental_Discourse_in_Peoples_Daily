# Scripts

These scripts document the project workflow:

- `crawler/`: People’s Daily collection scripts.
- `content_analysis/`: LLM-assisted coding script. Raw input files are not included in this repository; configure local input paths before running.
- `discourse_analysis/`: R scripts used for temporal summaries and visual analysis.

Do not commit API keys. Set `DASHSCOPE_API_KEY` through the environment when running the content-coding script.
